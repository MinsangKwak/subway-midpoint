# toss-subway-midpoint
A Toss WebView mini-app that calculates the optimal subway midpoint for multiple users based on their departure stations.


---

## ✨ 현재 구현 상태

- 웹 환경에서 동작하는 **Toss 스타일 UI** 구성
- TDS Mobile 디자인을 참고한 Button 컴포넌트 구현
- 출발지 입력 필드 및 CTA 흐름 구성
- 향후 Toss 인앱(WebView) 환경을 고려한 구조 설계

> ⚠️ 현재는 **웹 개발 환경** 기준으로 구현되어 있으며,  
> Toss 인앱 전용 TDS 컴포넌트는 추후 분기 적용 예정입니다.
